# Thuật Toán Theo Nhóm

Thư mục này mô tả các thuật toán hiện có trong simulator, tách theo 3 phần chính:

- `load_balancer.md`: thuật toán chọn worker trong load balancer.
- `service_models.md`: thuật toán/mô hình tính service time ở worker.
- `load_balancer_control.md`: thuật toán điều khiển tham số load balancer từ controller.

Các mô tả ở đây bám theo implementation hiện tại trong `lb_simulation/`.
